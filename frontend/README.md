# Frontend da Confeitaria

Interface do sistema. Feito em React.

Precisa do backend rodando na porta 3000 pra funcionar direito.

---

## Pra rodar

Entra na pasta frontend e roda:

```
npm install
npm run dev
```

Abre http://localhost:5173

O backend tem que estar ligado tambem. Na pasta backend:

```
npm run start:dev
```

---

## Telas

- Dashboard (graficos e numeros)
- Clientes
- Categorias
- Produtos
- Funcionarios
- Vendas

---

## Pastas

- api - chamadas pro backend
- pages - as telas
- components - menu lateral
- types - tipos do typescript

---

## Se der erro

Geralmente é porque o backend nao ta rodando ou o banco nao conectou. Testa no swagger: http://localhost:3000/api
